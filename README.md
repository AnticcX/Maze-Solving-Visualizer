# How To Run Program
To run the program, execute `python main.py` from its working directory

or

run `main.exe` located in the `/build/main` directory of this project

# How to use the program
Rows and Columns can be inputted through the text boxes. Minimum size is `5x5`, Maximum size is `999x999`.
Speed can be inputted through the text box. Minimum speed is `1`, maximum speed is `150`.
Use the buttons to swap between DFS and BFS, generate new randomized mazes, block the exit path, and running the simulation.

# Features
- Visualization of BFS maze solving
- Visualization of DFS maze solving
- Adjustable simulation speed
- No path test case
- Randomized maze generated with randomized DFS (recursive backtracing)
